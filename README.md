# Pic JS

## Initial Setup

Install dependencies:

```bash
bun i
```

## Examples

### Counter

Start DFX:

```shell
dfx start --background
```

Create a new canister:

```shell
dfx canister create counter
```

Build the counter canister:

```shell
bun build:counter
```

Run the counter tests:

```shell
bun test:counter
```
